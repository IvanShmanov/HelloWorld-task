Что бы скачать с гитхаба проект необходимо в терминале git Bash  перейти в папку в которой вы хотите разместить проект и вызвать команду 
	git clone https://github.com/IvanShmanov/HelloWorld-task.git
или скачать .zip файл и разархивировать его в эту папку.

Перейдите в папку HelloWorld-task\src\main\resources и посмотрите в файле database.properties параметры подключения к бд.
При необходимости измените их. Сделайте тоже самое в файле pom.xml в разделе <properties>. pom.xml находится в папке HelloWorld-task.
Затем из командной сторки переходим в папку HelloWorld-task.
После этого запускаем команду mvn clean package для сборки проекта в war файл.
Теперь запустим поочерёдно 2 команды: 
	mvn sql:execute@create-database 
Эта создаёт базу данных в postgreesql. Необходимо иметь стартовую базу данных postgres чтобы команда сработала.
	mvn sql:execute@insert-data 
Вызывает скрипт который создаёт таблицу и заполняет её.  
Выполним команду 
	mvn wildfly:run 
команда производит деплой проекта на сервер и запускает сервер.
Ждём когда запустится сервер.После запуска заходим в браузер по адресу:
	http://localhost:8080/HelloWorld
Видим результат.
